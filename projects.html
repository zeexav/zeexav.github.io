Project Summary

Traffic Violations ETL repo

This is an ETL Pipeline to explore the relationship between traffic camera violations in Chicago and the economic make-up of the community they are located in. The economic factors considered were median home vaue and median household income.

The traffic violations data was sourced from Chicago Open Data and the economic data from USZipcodes API. The datasets were both cleaned, transformed and then merged using the zipcode. Several views were created in MySQL to easily complete analysis on various combinations of the data.

Mars Project: Web Scraping Repo

Using BeautifulSoup and Requests/Splinter, I scraped the NASA Mars news site and stored the information in a MongoDB database. Each time the app is run, the data is scraped and stored in the database. The web page is populated by querying the database and loading the results into the web page.
